# Java Lab Work 3

## Topic
Working with One-Dimensional Arrays in Java

## Variant
Variant 1

## Task
In a given integer array X consisting of 20 elements:
- Determine and display the first negative element
- Display its position
- If all elements are positive, display a message

## Implemented Features
- Array input using Scanner
- Array formatted output (4 rows × 5 columns)
- Search algorithm for first negative element
- Object-oriented structure using previous Doctor class

## How to Run

Compile:
javac src/*.java

Run:
java -cp src LabWork3
